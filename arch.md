## вҡЎ 3. INTERNAL AI MODULE STRUCTURE

```
backend/
 в”ңв”Җв”Җ ai/
 в”Ӯ   в”ңв”Җв”Җ analyze.js        вҶҗ MAIN PIPELINE
 в”Ӯ   в”ңв”Җв”Җ tagging.js        вҶҗ image вҶ’ tags
 в”Ӯ   в”ңв”Җв”Җ caption.js        вҶҗ image вҶ’ caption
 в”Ӯ   в”ңв”Җв”Җ moderation.js     вҶҗ safety check
 в”Ӯ   в””в”Җв”Җ hashtags.js       вҶҗ tags вҶ’ hashtags
```
